Blog platform

Petri Ojala, Apr 5, 2021, Blog Github

Wouldn’t dance videos on Tik Tok be more like 2021 than a blog?

Maybe, or at least I could start to call myself a social media influencer but not sure if I would find the content that useful in the long run.

There’s probably a billion ways to run a blog nowadays, from legacy WordPress to some python functions in AWS Lambda but I’ve decided to do something different. I don’t want to maintain anything and there was a few things I wanted to try out anyway, so..

![Gopher)[/images/gopher.png)

This web site is built with Hugo, one of the most popular open-source static site generators or at least that’s what they say at https://gohugo.io/. It’s written in go so it must be..

The theme is Horthernlab by Themefisher, https://themes.gohugo.io/northendlab-hugo/ that is powered by Bootstrap 4 and responsive ready – seems to work fine on mobile browsers, too. I might switch to some more featurerich theme at some point though, we’ll see.

The source for the site is hosted on a private repository on Github. Whenever the content is updated, a Github Actions workflow will be triggered that will build the website with hugo. The resulting site is then uploaded to another, public Github repository that is available as Github Pages site with custom domain. All that happens in about 10-15 seconds, isn’t that cool?
